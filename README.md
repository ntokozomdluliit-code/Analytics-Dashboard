 ## 📊 Features

  - **Real-time Event Monitoring** - Live WebSocket stream of user interactions

  - **Threat Intelligence Integration** - Pulls data from free threat feeds
  - **Live Dashboard** - Visual monitoring with instant updates
  - **Event Tracking** - Page views, clicks, form submissions, Turnstile events

  - **Threat Detection** - Identifies phishing sites, fake logins, scam domains

  - **Multiple Data Sources** - Aggregates from Phishing.Database & Criminal IP
